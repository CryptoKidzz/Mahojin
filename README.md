### Mahojin-BOT

## Features
- Auto claim point
- Auto generate image
- Simple output

## Requirements
- Node.js 18+
- Npm (node package manager)

## Installation
1. Clone this repository
```bash
git clone https://github.com/CryptoKidzz/Mahojin.git
```
2. Install dependencies
```bash
npm install
```
3. Create screen
```bash
screen -S mahojin
```
4. Fill in your token in `data.txt`

5. Run 
```bash
node index.js
```
